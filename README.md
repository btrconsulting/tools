## Como usar makefile
- instalacion completa `make install all`
- instalacion por herramienta 'make install _herramienta_'

## como usar webmap

`docker run -d --name webmap -h webmap -p 8000:8000 -v /tmp/webmap:/opt/xml rev3rse/webmap`
